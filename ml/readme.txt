[[머신러닝]]

1.개요  
    -Machine Learning
    -데이터에서 지식을 추출하는 작업 : 데이터 - 정보 - 지식 - 지혜
    -통계학, 컴퓨터 과학, 데이터 마이닝, 인공지능 등 전문분야가 섞여있음
    -예측 분석, 통계적 머신러닝
    -예) 영화추천
2.정의
    -아서사무엘 : 명시적인 프로그래밍 없이 컴퓨터가 학습을 하는 능력
    -톰미첼 : 어떤 작업T에 대한 컴퓨터 프로그램의 성능P로 측정하였을 때, 경험E으로 인해 성능이
    향상되었다면, 이 컴퓨터 프로그램은 작업 T와 성능측정 P에 대해 경험E로 학습한 것이다.
3.데이터 과학(분석, 모델링 등 ) 절차
    1) 연구 목표 설정
        -전체 단계를 보고 주제, 목적, 산출물의 형태를 고려
        -무엇을 분석, 예측모델, 평가방법
        -단계별 산출물의 형태를 정하고 들어가는 것이 중요
    2) 데이터 획득
        -lv1 : 제공받는다 - 연구용 데이터, 경쟁, 공공데이터, 정부기관요청, 데이터 바우처
        -lv2 : open API (데이터를 쿼리를 통해 제공)
        -lv3 : 웹 스크래핑 (접근이 용이)
        -lv4 : 크롤링 (ajax, 인증, 검색 등 사람의 손을 타야하는 사이트)
        자동화, 스케줄러(window-작업스케줄러, linux-cron)
    3) 데이터 준비
        -데이터 품질 향상 목표
        -데이터 정제, 통합, 변환 -> 품질 향상 목표
        -모델에 적합한 형태로 데이터를 변환
    4) 데이터 탐색
        -데이터 이해하는데 목적 -> 통찰
        -통계적 분석(통계 요인을 이용), 시각적 분석(eda)
    5) 데이터 모델링 및 모델 구축
        -통계적 모델링
        -머신러닝 모델링(세부단계가 더 존재)-딥러닝 모델링
    6) 시스템 통합
        -서비스, 모듈, 라이브러리, 보고서
        -이 부분은 1단계에서 이미 결정된 부분
4. 머신러닝 시스템 종류
    -지도 학습 : supervised learning
        -알고리즘에 주입하는 훈련 데이터에 레이블(label, 정답, class)이 포함
        -분류, 회귀
    -비지도 학습 : unsupervised -
        -훈련 데이터에 레이블이 없다
        -알고리즘은 시스템의 도움 없이 학습을 진행
        -군집, 차원축소, 시각화, 연관규칙
    -준지도 학습 : semi-
        -레이블이 일부만 존재
        -통상적으로 레이블이 없는 데이터가 다수
        -심층신뢰신경망..
    -강화 학습 : reinforcement -
        -에이전트 : 학습하는 시스템
        -환경관찰, 행동을 실행, 이에 따른 결과를 보상(긍정적보상-reward,부정적보상-penalty)부여
        -에이전트는 가장 큰 보상을 얻기 위한 최적의 정책, 전략을 스스로 학습 ex)알파고
